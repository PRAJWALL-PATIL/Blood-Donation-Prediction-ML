# Blood Donation Prediction using Machine Learning

This project predicts whether a donor will donate blood in the next university blood donation campaign using Machine Learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Imbalanced-learn (SMOTE)
- Jupyter Notebook

## Machine Learning Model
**Random Forest** (Best performing model)

## Project Workflow
1. Data Cleaning  
2. Exploratory Data Analysis (EDA)  
3. Outlier Detection & Treatment (IQR Method)  
4. Feature Engineering & Correlation Analysis  
5. Data Scaling (StandardScaler)  
6. Handling Class Imbalance (SMOTE)  
7. Model Training (Random Forest, SVM, Decision Tree, KNN)  
8. Hyperparameter Tuning  
9. Model Evaluation

## Results
The **Random Forest** model predicts whether a donor will donate blood in the next campaign and performs well using Accuracy, Precision, Recall, and F1-score evaluation metrics.
